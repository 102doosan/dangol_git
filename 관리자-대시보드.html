📊 관리자 대시보드
a-dashboard · 필수 · FE/BE
화면 구성
상단 헤더: 날짜·미처리 건수 요약 + 미니 통계 3개 (전체 회원·미처리·이번달 매출)
요약 카드 4개: 처방 신청 대기 · SFMA 도래 · 신규 회원 · 레슨 미배정
좌측 팝업 알림: 탭(전체·처방 신청·SFMA·레슨) + 미처리 알림 목록 (색상 dot로 유형 구분)
우측 미처리 업무: 알림 목록의 미완료 건과 연동 — 카테고리별 섹션 구분 + 체크박스
하단 매출 차트: 월별 바 차트 (6개월 기본) + 회원 현황 도넛 차트
동작
알림 항목 클릭 → 해당 관리 화면으로 이동 (처방 신청 → My 운동처방 관리, SFMA → SFMA 관리)
미처리 업무 체크박스 → 완료 처리 시 알림 목록에서도 제거
알림 탭 전환 → 해당 유형 알림만 필터링
매출 차트 기간 버튼 → 3개월 / 6개월 / 1년 전환
API
GET /admin/dashboard/stats — 요약 카드 수치
GET /admin/notifications?status=unread — 미처리 알림 목록
GET /admin/sales?period=6m — 매출 차트 데이터
PATCH /admin/notifications/:id/read — 알림 읽음 처리
